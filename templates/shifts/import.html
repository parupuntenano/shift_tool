マスタ・スキル表を取り込む

.xlsx / .xls / .csv に対応しています。

{% csrf_token %} {{ form.as_p }}
必要な見出し 社員番号, 氏名, 公休数, 備考 迷ったら右上の「見本フォーマット」をダウンロードして、その形のまま編集してください。 動作確認だけしたい場合は「サンプルデータ」を取り込むと、スタッフ10人・業務3つをまとめて試せます。 公休数はスタッフ管理へ自動反映します。希望上限はExcelに書かず、スタッフ管理の一括変更で管理します。 「業務マスタ」シートに業務名・必要人数・色・有効を入力すると、業務管理へ自動反映します。色はコード入力のほか、業務名セルの塗りつぶし色からも読み取れます。 業務スキルも取り込む場合は、公休数・備考の後ろに業務マスタと同じ業務名を見出しとして追加し、セルにスキル記号を入力します。 見本の「スキル区分」シートに記号・意味・優先度・アサイン可を書くと、会社ごとのマーク設定も自動反映します。 備考には「2勤1休」「4勤不可」「単休不可」「ロールとエーカス交互」「ロール禁止」などを書くと、個別制約へ自動反映します。 初回取込時は、ログインIDを社員番号、初期パスワードを「0000」として自動登録します。

先月シフト実績を取り込む

マスタ取込後に、先月の実績表を取り込んで月またぎの制約判定に使います。

{% csrf_token %} {{ previous_shift_form.as_p }}
必要な見出し 社員番号, 氏名, 2026/6/24, 2026/6/25 ... Excelの場合は「先月シフト実績」シートだけを読み込みます。別シート名だと取り込めません。 指定した月の月末7日分だけを読みます。例：6月なら 2026/6/24〜2026/6/30、7月なら 2026/7/25〜2026/7/31。 日付列は「24」「6/30」「2026/6/30」のように入力できます。 セルが業務名なら勤務、休/公休なら公休、有休/有給なら有給として取り込みます。 業務名は先に取り込んだ業務マスタと一致させてください。
