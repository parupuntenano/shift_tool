# Shift Tool V2

30人規模の現場向けに絞り直した、管理者専用の月間シフト作成システムです。

## 方針

- スタッフログインなし
- Excel取込を起点にする
- 必要人数を優先して自動作成する
- 公休数や不足は警告で見える化する
- 最後は管理者が画面で手直しする
- Excel/CSVで出力する

## 起動

```powershell
cd C:\Users\haruh\OneDrive\デスクトップ\シフト作成ツール\shift_tool_v2
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver 8002
```

ブラウザで `http://127.0.0.1:8002/` を開きます。

## Excel構成

V2は以下の4シートを読みます。

- `スタッフ`
  - 社員番号、氏名、公休数、その他
- `業務`
  - 業務名、必要人数、有効
  - 業務名セルの塗りつぶし色を業務色として取り込みます
- `スキル`
  - 社員番号、氏名、各業務のスキル記号
  - 初期記号は `◎` 指導可能、`○` 配置可、`△` 研修中、`×` 不可
- `前月実績`
  - 社員番号、氏名、日付列
  - セル値は業務名、公休、有休、余剰を読みます
